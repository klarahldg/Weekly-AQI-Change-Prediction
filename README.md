# Video Analysis: Hong Kong Protests

## Description
This project analyses visual patterns in news coverage of the Storming of the Legislative Council (2019).

## Methods
- Shot detection (SSIM)
- Colour analysis (KMeans)
- Movie barcodes

## Structure
- src/: methods
- notebooks/: analysis
- data/: video corpus

## Usage
Run notebooks in order:
1. exploration.ipynb
2. analysis.ipynb
